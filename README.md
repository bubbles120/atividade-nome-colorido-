# Nome Colorido

Projeto web simples e divertido que exibe cada letra do seu nome em uma cor diferente, com animações.

## Como usar

1. Coloque os três arquivos na mesma pasta:
   - index.html
   - style.css
   - script.js
2. Abra o arquivo `index.html` no navegador.
3. Digite seu nome no campo de texto.
4. Escolha um efeito de animação.
5. Clique em "Novas cores" para embaralhar a paleta.

## Efeitos disponíveis

| Efeito     | Descrição                          |
|------------|------------------------------------|
| Saltitar   | Letras sobem e descem em onda      |
| Onda       | Letras crescem e diminuem          |
| Pulsar     | Letras piscam suavemente           |
| Tremer     | Letras tremem para os lados        |
| Arco-íris  | Cores giram pelo espectro de cores |
| Sem efeito | Letras estáticas                   |

## Tecnologias

- HTML5
- CSS3
- JavaScript puro (sem bibliotecas)

## Compatibilidade

Funciona em qualquer navegador moderno diretamente pelo arquivo local (sem necessidade de servidor).
